# B64-URL-Decoder
> B64 URL Decoder es una extensión que funciona como herramienta de descifrado.
> 
> [!CARACTERÍSTICAS]
> - Detección de códigos/URL escriptados Base64.
> - Soporte de páginas: modo global y exclusivo.
> - Temas: oscuro/negro (sistema).
